# @lumina-study/graph

Graph library for Lumina Study

## Installation

```bash
pnpm add @lumina-study/graph
```

## Usage

```typescript
import { hello } from '@lumina-study/graph';

console.log(hello('World')); // Hello, World!
```

## Development

### Install dependencies

```bash
pnpm install
```

### Run tests

```bash
pnpm test
```

### Build

```bash
pnpm build
```

### Lint

```bash
pnpm lint
```

### Release

```bash
pnpm release
```
